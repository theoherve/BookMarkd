# FAQ - Installer BookMarkd sur mobile

## Comment installer BookMarkd sur mon téléphone ?

BookMarkd peut être installé comme une application sur votre téléphone Android ou iPhone, pour un accès rapide et une expérience optimale.

### Sur Android (Chrome)

1. Ouvrez BookMarkd dans Chrome
2. Attendez quelques secondes - un bouton "Installer BookMarkd" apparaîtra en bas de l'écran
3. Appuyez sur "Installer"
4. Confirmez l'installation dans la fenêtre qui s'affiche
5. BookMarkd sera ajouté à votre écran d'accueil

**Alternative :**
- Ouvrez le menu Chrome (⋮) > "Ajouter à l'écran d'accueil"
- Ou allez dans les paramètres Chrome > "Ajouter à l'écran d'accueil"

### Sur iPhone/iPad (Safari)

1. Ouvrez BookMarkd dans Safari
2. Appuyez sur le bouton de partage (□↑) en bas de l'écran
3. Faites défiler et sélectionnez "Sur l'écran d'accueil"
4. Personnalisez le nom si vous le souhaitez
5. Appuyez sur "Ajouter"

### Sur ordinateur (Chrome/Edge)

1. Ouvrez BookMarkd dans Chrome ou Edge
2. Cliquez sur l'icône d'installation (⊕) dans la barre d'adresse
3. Confirmez l'installation
4. BookMarkd s'ouvrira dans une fenêtre dédiée

## Pourquoi installer BookMarkd ?

- **Accès rapide** : Lancez BookMarkd directement depuis votre écran d'accueil
- **Mode hors ligne** : Consultez vos listes et livres même sans connexion internet
- **Expérience optimisée** : Interface adaptée mobile, sans barre d'adresse
- **Notifications** : Recevez des alertes sur les nouvelles activités (bientôt disponible)

## Le bouton d'installation n'apparaît pas

Plusieurs raisons possibles :

- **Déjà installé** : Si BookMarkd est déjà installé, le bouton ne s'affiche pas
- **Navigateur non compatible** : Utilisez Chrome (Android) ou Safari (iOS)
- **Connexion HTTPS requise** : BookMarkd doit être accessible en HTTPS (ou localhost)
- **Vous avez masqué le prompt** : Le bouton peut être masqué. Rechargez la page ou effacez les données du site dans les paramètres du navigateur

## Puis-je utiliser BookMarkd hors ligne ?

Oui ! BookMarkd fonctionne hors ligne :

- **Consultation** : Consultez vos listes, livres et avis déjà chargés
- **Actions en attente** : Likez, notez, ajoutez des livres - vos actions seront synchronisées automatiquement à la reconnexion
- **Page offline** : Si une page n'est pas disponible, une page de secours s'affiche avec un bouton de rechargement

## Comment désinstaller BookMarkd ?

### Android
- Maintenez l'icône appuyée > "Désinstaller" ou "Supprimer"

### iPhone/iPad
- Maintenez l'icône appuyée > "Supprimer l'application"

### Ordinateur
- Chrome/Edge : Paramètres > Applications > BookMarkd > "Désinstaller"

## BookMarkd prend-il beaucoup d'espace ?

Non, BookMarkd est une Progressive Web App (PWA) légère. Elle utilise principalement le cache du navigateur pour stocker temporairement vos données. L'espace utilisé est minimal (quelques Mo).

## Les notifications fonctionnent-elles ?

Les notifications push sont en cours de développement. Elles seront disponibles prochainement pour vous alerter sur :
- Nouvelles activités de vos amis
- Demandes de suivi
- Likes sur vos avis
- Recommandations personnalisées
