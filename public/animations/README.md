# Animations Lottie

Ce dossier contient les fichiers d'animation Lottie utilisés dans l'application.

## Book Loader Animation

Pour utiliser l'animation de livre pour le loader :

1. Allez sur https://lottiefiles.com/free-animation/book-loader-IRPpe2DJQ1
2. Cliquez sur "Download" et sélectionnez "Lottie JSON"
3. Enregistrez le fichier sous le nom `book-loader.json` dans ce dossier (`public/animations/`)

Le composant `BookLoader` chargera automatiquement cette animation.
